  SmartLibrary API

.NET RESTful API, предназначен за управление на отдаване под наем на книги в библиотечна система. Този проект демонстрира N-слойна архитектура, инжектиране на зависимости и имплементация на бизнес логика, следвайки принципите на чист код.

 Преглед на проекта

Системата позволява на потребителите да:
1. Регистрират (симулирано чрез seed/creation).
2. Добавят книги към инвентара.
3. Отдават книги под наем (проверка на наличността от бизнес логика).


---

 Архитектура

Решението е разделено на 4 слоя:

1. SmartLibrary.Domain: Основни обекти (`Book`, `User`). Няма зависимости.

2. SmartLibrary.Application: Интерфейси, DTO, Валидатори, Услуги (`LendingService`).

3. SmartLibrary.Infrastructure: Имплементация на база данни (Mongo Driver), Хранилища.

4. **SmartLibrary.Api**: Контролери, Middleware, DI конфигурация.
